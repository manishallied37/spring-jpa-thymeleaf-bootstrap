The Building Blocks APIs are a set of common APIs which address ancillary requirements for integration to various processes. These set of APIs provide capabilities like retrieve credit rating scores, validation services to establish authenticity, eligibility checks, list of available Pre approved offers and multiple other common capabilities. These can be integrated with ease into your API sequence and be useful in the execution of the multiple business use case across various processes, products and solutions.
