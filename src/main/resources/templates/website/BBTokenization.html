These are the set of APIs related to Tokenization operations

Product API Description
Approve Tokenization This api is uset to Approve Tokenization
Token Inquiry by PAN This API is used to retrieve all tokens related to a given PAN
Get Verification Methods This API allows the issuer to provide supported methods of cardholder step-up authentication
Send Activation Code This API is invoked to deliver the OTP to the destination selected by the cardholder, such as an email address, via the issuer.
Token Created Notification This API is used to invoke the API call when the issuers token has been created.
Token Updated Notification This API is used to invoke the API call when the issuers token has been updated
Device Token Binding Request This API is invoked to bind an ECOM/COF token to a device.
Token Lifecycle (TLCM) This API enables the issuer to activate, resume, suspend, or delete a card token.
Pan Lifecycle This API enables the issuer to update PAN and PAN expiration date. It also helps perform VAU updates.
Token Inquiry This API is used to retrieve token details for a particular token, including device and risk information.
