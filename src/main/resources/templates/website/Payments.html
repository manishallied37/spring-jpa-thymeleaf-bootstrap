Leverage the set of payments API in the domain of UPI payments, NEFT or IMPS and others, to create payment solution along with ICICI Bank. The payment APIs enable the complete payment lifecycle from registration to making transactions, checking status and de-registering. Merchants and organizations that need to place bulk remittance transactions and track their status, we provide solutions to ease the pain. These robust APIs facilitate you to traverse across this process with ease whether it is- placing requests, facilitating payments, tracking requests or publishing consolidations. So, whether it’s an organization, institution or cooperative society, these API services facilitate easy creation of the needful for bulk remittances and their tracking.
