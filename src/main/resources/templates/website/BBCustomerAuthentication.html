Leverage these APIs for customer authentication based on mobile number or customer account etc.

Product API Description
OTP Generation This API will be used to create the OTP and the OTP will be send to customer Mobile Number.
TransUnion(Canada) This api is used to capture customer details and upload it in transunion
Customer AML Check(Fircosoft_OSB) This API is used to check whether the customer is having any AML propensity or not
Customer AML Check(Fircosoft_OSB) Callback This API is used to check whether the customer is having any AML propensity or not.This will send the response of Customer AML Check(Fircosoft_OSB) API as call back
DIY-CIBIL This API is used to fetch the CIBIL score range of the customer
Fetch PAN details This API will fetch PAN details of the customer
OTP Verification This API will be used to verify the OTP.
Aadhar Based OTP Generation This API is used to generate OTP for Aadhar
Aadhar Based OTP Verification This API is used for OTP verification for Aadhar.
Customer Handshake This service is used to verify the customer details
WaitCallTime This api gives callwaittime details in response based on groupid
Get Details By TrackingId This api gives the status details based on tracking ID on groupid
ekyc_updation This API will be used for submission of an Original Transaction to internal system.
ShuftiPro For Customer online KYC verification we have called Shufti Pro API.
ShuftiPro_Delete This api is for deleting KYC record in sufti pro
ShuftiPro_Callback In Shufti Pro Callback URL we have to configured API Gateway (ICICI Bank) API URL so that after customer KYC verification Shufti Pro will send response to API Gateway Gateway.
