Credit Cards

  1. Loans And Cards
  2. Cards
  3. Credit Cards

Credit Cards API support the end to end customer journey, from initiating the customer request for credit cards, available cards based on customer profile , enabling request processing , credit card creation and services to manage credit cards. The key available APIs are:

Product API Description
OTP Creation This API is leveraged to create and deliver OTP as SMS for authentication or confirmation as required in the process.
Block Online Transactions This api is used to block the Online Txn on card.
Unblock Online Transactions This api is used to unblock the Online Txn on card.
Block ATM Withdrawals This API is used to block the ATM Cash withdrawal on card. No authorization header is required to invoke this api
Unblock ATM Withdrawals This api is used to unblock the ATM Cash withdrawal on card.
Block International Transactions This API is used to block the International Txn on card
Unblock International Transactions This api is used to unblock the International Txn on card.
Prime Credit Card This API is used for retrieving customer details
convertTransactionToEMI This API is Used to convert transaction in to EMI
CCAplicationStatus This API Fetches the status of Credit card application
Create CC Bill Payment This API is used to intiate credit card payment through DIY portal
Get Offer This API is leveraged to Avail all the Credit Card offers available to the customer on the basis of Linked Mobile Number after OTP verification
Fetch CC Bill Payment This API is used to fetch Credit card payment details
Card Creation This API is leveraged to create the card as the final acceptance step.
Card Status This API is used to check the status of card.
View Statement This API will be used for checking the mini statement of customer
Available Credit Limit This API is used to find out available limit of credit card.
View Reward Points This api is used to display Reward point balance on card fetched from PRIME.
Block Card This API is used to block the card temporary or permanently.
Unblock Card This API is used to unblock the cards.
